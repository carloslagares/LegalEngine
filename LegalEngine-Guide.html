What is LegalEngine?

AI compliance assistant, on demand

LegalEngine is a web application that answers questions about vacation rental regulations across Spain and Europe. It works like having a compliance expert on call: the user types a question, the system identifies the relevant region and topic, retrieves the appropriate legal guide, and uses Claude AI to generate a precise, sourced answer — streamed in real time.

🔍

Smart region detection

Automatically identifies which Spanish region the user is asking about and loads the right guide as context.

⚡

Real-time streaming

Answers appear token by token using Server-Sent Events (SSE), providing instant feedback while Claude generates the response.

📚

Grounded in real guides

Every answer is based on actual legal guides written by lawyers. The AI never invents requirements — it only uses what is in the knowledge base.

🔄

Expandable knowledge

Adding a new region or topic is as simple as dropping a Markdown file into the knowledge/ folder.

Important: LegalEngine provides operational intelligence based on curated legal guides. It is not a substitute for legal advice. Users should always verify critical information with a licensed lawyer.

How to use it

Three ways to interact

Method 1 — Type a question directly

Type any compliance question in the search box. If the question mentions a region, the assistant answers immediately.

Example questions

"What steps do I need to register a VUT in Madrid?"

"How many square meters per guest does Andalucia require?"

"Do I need to report guests to the police in Barcelona?"

"What is the tourist tax rate in the Balearic Islands?"

Method 2 — Click a category pill

The pills above the search box rotate between 7 categories. Clicking one fills the search box with a relevant sample question.

Regional Regulations Guest Registration & GDPR Identity Verification Rental License City Tax Electronic Invoicing Payment & Taxes

Categories in amber are newly added and rotate in every 6 seconds with a subtle fade animation.

Method 3 — No region? The assistant asks

If your question doesn't mention a specific region, a region picker appears with all available communities. Select one and the answer is generated with that guide as context.

Tip: If no region is selected and none is detected, the assistant sends all guides as context. This works well for comparative questions like "Which region has the fastest registration process?"

How to expand the knowledge base

Adding regions and topics

Option A — Add a new Spanish region

1

Drop the new PDF guide into Guias/

2

Add an entry to FILENAME_MAP in scripts/extract-guides.js with slug, name, and aliases

3

Run npm run extract to generate the Markdown and update _index.json

4

Commit and push — Railway auto-deploys in ~60 seconds

Option B — Add a new thematic guide

1

Create a .md file in knowledge/ with structured content

2

Add its entry to knowledge/_index.json with "type": "thematic" and relevant aliases

3

Add a regex trigger in THEMATIC_TRIGGERS in server.js for auto-detection

4

Optionally add a new pill category in allPillCategories in index.html
